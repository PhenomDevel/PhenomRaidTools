API:SendMessage (type, targets)
Send a message to the given players. The message will be sent to all players in `targets` and it will perform the corresponding action based on `type`.

Parameters:

  • type type of the message you want to send
  • targets table of targets which should receive the message

Returns:

    true when message could be sent

Usage:

    _G["GPRTAPI"]:SendMessage("advanced", {"Phenom"})
API:AddGlobalPlaceholder (placeholderType, name, playerNames)
Add a new global placeholder

Parameters:

  • placeholderType the type of the placeholder. Can either be `group` or `player`
  • name the name of the placeholder. Can only be a name which is not already present
  • playerNames table of the player names which should be used for the placeholder

Returns:

    true when placeholder could be created

Usage:

    _G["GPRTAPI"]:AddGlobalPlaceholder("player", "test", {"Phenom"})
API:AddEncounterPlaceholder (encounterID, placeholderType, name, playerNames)
Add a new encounter placeholder

Parameters:

  • encounterID the encounter id for which the placeholder should be added
  • placeholderType the type of the placeholder. Can either be `group` or `player`
  • name the name of the placeholder. Can only be a name which is not already present
  • playerNames table of the player names which should be used for the placeholder

Returns:

    true when placeholder could be created

Usage:

    _G["GPRTAPI"]:AddEncounterPlaceholder(1234, "player", "test", {"Phenom"})
